================================================================================
                  CRM 客户关系管理系统
                  (基于 Qt 6 + SQLite)
================================================================================

一、项目简介
--------------------------------------------------------------------------------
本系统是一款专为销售团队设计的轻量级客户管理与跟进平台，采用 Qt 6.5 + C++20
开发，底层使用 SQLite 数据库实现单文件本地存储，方便数据交接与迁移。


二、核心功能
--------------------------------------------------------------------------------
1. 智能权限管理
   - 根据用户角色（销售/经理/管理员）自动过滤数据视图
   - 严格的权限控制确保数据安全

2. 全生命周期客户管理
   - 从线索录入到成交客户全覆盖
   - 完整的客户信息管理

3. 跟进记录与时间线
   - 支持客户沟通历史记录
   - 动态时间线展示跟进进度

4. 公海池机制
   - 支持客户认领功能
   - 自动回收长期未跟进客户

5. 人事管理（管理员权限）
   - 员工部门分配
   - 权限变更管理
   - 一键交接离职员工资产

四、技术栈
--------------------------------------------------------------------------------
- 开发框架：Qt 6.5
- 编程语言：C++20
- 数据库：SQLite（单文件本地存储）
- 构建工具：CMake 3.19+
- 核心模式：面向接口编程（ICustomerRepository）

六、构建说明
--------------------------------------------------------------------------------
1. 环境要求
   - Qt 6.5 或更高版本（推荐 Qt 6.8+）
   - CMake 3.19 或更高版本
   - 支持 C++20 的编译器

2. 构建步骤（Windows）
   cd build
   cmake .. -G "MinGW Makefiles"  # 或使用 Qt Creator / Visual Studio
   cmake --build . --parallel

3. 构建步骤（Linux/macOS）
   mkdir build && cd build
   cmake ..
   cmake --build .

4. 运行程序
   构建完成后，可执行文件位于 build 目录


七、开发约定
--------------------------------------------------------------------------------
1. 数据交互必须经过 ICustomerRepository 接口，严禁前端直接操作数据库
2. 所有实体类使用 const 引用传参，提升系统性能
3. 头文件放在 SourceCode/include/ 目录
4. 实现文件放在 SourceCode/src/ 目录
5. UI 文件放在 SourceCode/src/ 目录
